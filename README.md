# SC-302-Documentacion-del-Software - ODS 14: Vida Submarina

## 📝 Descripción del Proyecto
Este proyecto fue desarrollado como el trabajo final para el curso **Documentación de Software**. Consiste en una solución tecnológica conceptual y práctica enfocada en abordar las metas del **Objetivo de Desarrollo Sostenible 14 (Vida submarina)** de la ONU, orientándose en **MarAlert** que es una aplicación móvil de concientización sobre la vida marina. 

Ofrece guías de cuidado del medio acuático, protocolos de actuación ante emergencias (derrames, pesca ilegal, contaminación química) y una plataforma de reporte ciudadano para proteger mares, ríos y lagos de Costa Rica.

El objetivo principal es aplicar las mejores prácticas de la industria en cuanto a documentación interna/externa, modelado de sistemas (UML) y metodologías ágiles (SCRUM) para asegurar un ciclo de vida de desarrollo de software (SDLC) robusto, escalable y de alta calidad.
